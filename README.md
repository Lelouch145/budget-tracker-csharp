# Budget Tracker – C#

A simple console-based budget tracker built in C#.  
The application allows users to track income and expenses directly from the terminal.

This project was created to practice fundamental programming concepts in C#, including object-oriented programming, lists, loops, and input validation.

---
## Features

- Add income
- Add expenses
- View all transactions
- Calculate and display total balance
- Input validation för transaction name and amount

## Concepts Used

This project demonstrates the use of several core C# concepts:

- Classes and objects
- Lists (`List<T>`)
- Methods
- - Loops (`for`, `foreach`)
- Conditional statements (`if`, `switch`)
- Input validation
- Parsing user input using `double.TryParse()`

---
## Project Structure

The project contains a simple structure:

**Program.cs**  
Contains the main application logic.

**Transaction class**  
Represents a single transaction with the following properties:

- Name
- Amount
- Type (Income or Expense)
